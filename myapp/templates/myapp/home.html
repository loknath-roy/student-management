What is Student Management System ?


A Student Management System is designed to record,analyze,and manage information in a school. These systems are updated by teachers and school administrators on a rolling basis to batter serve the need of the greater student body.

Without a Student Management System,schools would become disorganized,staff would lack clarity into scheduling and student activity, and it would become increasingly difficult for school districts to leverage data in the dicision-making process.

Purpose of Student Management System


The school/college management has to handle records fo many number of students and maintenance was difficult.Though it has used an information system,it was totally manual.Hence there is a need to upgrade the system with a computer based information system.

The Student Management System allows authorized users to access the record of academically registered students.It can be used in various educational institutes across the globe and simplifies working of institutes.

This system helps in maintaining the database of the students in any educational organization.We can easily access any students information anytime and can be kept safely for long period of time without any damage.





Features of a Student Management System


  • CREATION OF NEW RECORD : This function creates a record for a new student.

  • DISPLAY OF DATA IN RECORD : This function displays the record of the students.

  • UPDATION IN RECORD : This function updates the information in a record of any student.

  • DELETION OF RECORD : This function is used to delete the existing record of any student.
